# igornovaeslins.github.io

Personal academic website of Igor Novaes Lins — political scientist (criminal governance, elections, and political violence in Latin America).

Live at **https://igornovaeslins.github.io**

Static single-page site (plain HTML/CSS, no build step). To edit, change `index.html` and push to `main`.

SEO assets: `sitemap.xml`, `robots.txt`, Open Graph tags, and schema.org `Person` structured data are all in place. When verifying the site in Google Search Console, paste the verification meta tag into the marked spot in `index.html` (`ADD-GOOGLE-VERIFICATION-HERE`).
